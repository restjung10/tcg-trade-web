목표 : 사기당할 일 없는 tcg카드 거래 커뮤니티 만들기

필요 기능 :
카카오 로그인 연동(1인 1계정을 위함)
판매글 작성 시 이미지 필수 첨부
첨부 된 이미지 AI생성 이미지 검증 기능
검증 후 닉네임/날짜/시간 워터마크 자동 부여
거래를 위한 1:1 채팅 기능 구현
판매, 구매 등 게시판 세분화
전체적인 디자인은 DCinside의 느낌을 따라가는 쪽으로

후에 추가할 기능 :
계좌 인증 시스템 (사용자가 마이페이지에 계좌번호를 작성하면 운영자가 터치트 등을 확인 후 승인/거절 하는 제도)
거래 성사 시 해당 게시물 완료 게시판 등으로 이동하여 거래 된 게시글과 안된 게시글 구분

이 프로젝트는 개인 취미로 시작하는 것이므로
예산을 최소화 하는 방향으로 진행
또한 웹 사이트에 보안을 철저히 지킬 수 있는 방향으로 제작 바람
